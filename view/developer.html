Web Service Documentation

The Joblint web service offers a simple way to lint your job specs without installing on your local machine. There is a single endpoint, and there aren't any rate limits (at the moment) â€“ just be nice please :)

Request

POST http://joblint.org/ws HTTP/1.1
User-Agent: MyApp/1.0 (a unique identifier for your application)
Content-Type: application/json

{spec:"Your job spec goes here"}

Response

HTTP/1.1 200 OK
Content-Type: application/json; charset=utf-8

{
    "failPoints": {
        "culture": 0,
        "realism": 0,
        "recruiter": 0,
        "tech": 0
    },
    "errors": [ ... ],
    "warnings": [ ... ],
    "notices": [ ... ]
}
